# Style Guide: pointrr.dev (Dark Mode SaaS)

## Visual Identity
The UI should mirror a "Linear/Vercel" aesthetic: Pure black backgrounds, subtle 1px borders, and high-vibrancy Indigo accents.

## Technical Constraints (daisyUI)
- **Theme Name:** Use the custom pointrr theme.
- **Backgrounds:** Use bg-base-100 for the page and bg-neutral for cards.
- **Buttons:** Primary buttons should be .btn-primary (Color: 
#5841FF). 
- **Typography:** - Headings: font-sans font-bold tracking-tight.
  - Code/Domains: font-mono text-sm.
- **Components:** - Use card-bordered with border-secondary.
  - Use backdrop-blur for navigation and dropdowns.

## UI Specifics
- **Grid Background:** Apply a radial-gradient dot grid to the main container.
- **Shadows:** Avoid heavy shadows; use thin borders (border-secondary) or subtle primary-colored glows for active states.
- **Layout:** High whitespace, centered hero sections, and clear "Flow" diagrams for domain routing.
